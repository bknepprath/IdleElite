Base Action Time base_seconds = (1.0 + stamina * 0.75 + unlock_level * 0.06) * skill_time_scale * time_mod `time_mod` is per action, so early actions across skills do not share the same duration.
Mastery Scaling seconds = max(base * 0.35, base * (1 - mastery_level * mastery_speed))
stamina = max(1, ceil(base_stamina * (1 - mastery_level * stamina_cut))) `mastery_speed` and `stamina_cut` are tunable per action and shown on each card.
Skill Level Scaling success = min(98%, base_success + max(0, skill_level - unlock_level) * success_growth) `success_growth` is per action, so some actions stabilize faster than others.
