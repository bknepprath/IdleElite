Core Loop

Skill-specific stamina creates natural movement. The player drains one activity pool, pivots to another, buys upgrades, and returns after regeneration or offline progress.

30 sec to 5 min sessions
1. Pick Skill

Choose the stamina pool that is ready or needed for a goal.

2. Run Activity

Spend stamina on a skill-specific action with unique timing, mastery, and success tuning.

3. Grow Mastery

Gain skill XP and per-activity mastery. Mastery makes that exact action faster and cheaper.

4. Rotate

When stamina is low, hop to another skill and keep momentum alive.

Activity Button Rules

The current prototype treats the activity button as the main game object. It must show title, reusable art, reusable background art, stamina, time, success, mastery progress, and the running progress rail without instructional clutter.

125 action review

Base Time

base_seconds = (1.0 + stamina * 0.75 + unlock_level * 0.06) * skill_time_scale * time_mod. Each action owns a time_mod, so level 1 actions across different skills do not take the same time.

Mastery

Each activity has its own mastery level. mastery_speed lowers action time and stamina_cut lowers stamina cost, both tuned per action.

Skill Level

Skill level improves success chance after unlock. success_growth is per action, so some actions become reliable faster than others.

Motivation Star

The hero randomly becomes motivated to do one accessible action for 5 completions. Motivated actions show a star and grant +15% XP plus 10% faster cycles until the charges are spent, then motivation rerolls.

Systems

The rebuild keeps the old game's bones: global level from skill levels, passive cash, and activity stamina. The new version adds identity rules, offline progress, and monetization-friendly reward boosts.

Global Level

Sum of all skill levels. Unlocks the activity log, titles, upgrade collections, offline multipliers, and prestige.

Stamina

Each skill has its own pool. Players rotate often because Fight being tired does not stop Fishing or Build.

Activity Mastery

Every individual action has its own mastery level. Repeating a specific action makes that action faster and lowers its stamina cost.

Rewarded Ads

Optional boosts: double offline earnings, refill one selected stamina pool, temporary success buff, or daily goal reroll.
